## What's Supported

### Supported Actions

This connector supports:

- [Proxy Actions](/proxy-actions), using the base URL `https://api.getgo.com`.

## Before You Get Started

To integrate GoToWebinar with Ampersand, you need to [Create a GoToWebinar Account](#create-a-GoToWebinar-crm-account) and obtain the following credentials from your GoToWebinar App:

- Client ID
- Client Secret

You will then use this credential to connect your application to Ampersand.

### Create a GoToWebinar Account

Here's how you can sign up for an GoToWebinar account:

1. Go to the [GoToWebinar Support](https://support.goto.com/contact) page.
2. Fill out the form to request a demo or contact GoToWebinar's sales team to set up an account.

### Creating an OAuth app in GoToWebinar

Once your GoToWebinar account is ready, you need to create an OAuth application to connect with Ampersand. Follow the steps below to create the OAuth app:

1. Log in to Your [GoToWebinar Account](https://identity.goto.com/login).

2. Go to [Developers](https://developer.logmeininc.com/clients).

3. Click on **Create a new client**

4. Go to the **OAuth Apps** tab.

5. Click the **Create App** button.

6. Enter the following details in the **Create OAuth App** form:

   1. **App Name**: The name of the OAuth app.
   2. **App Description**: Description for the OAuth app.
   3. **Redirect URL**: Enter the Ampersand redirect URL: `https://api.withampersand.com/callbacks/v1/oauth`

7. Click **Next**.

8. Select the applicable permissions.

9. Click **Save**.

You'll find the **Client ID** and **Client Secret** keys in your Oauth app details. These credentials are required for connecting your app to Ampersand, so be sure to note them carefully.

## Add GoToWebinar App Details in Ampersand

1. Log in to your [Ampersand Dashboard](https://dashboard.withampersand.com).

2. Select the project where you want to add the GoToWebinar App.

   ![Ampersand Project Selection](/images/provider-guides/090c05f-Ampersand.png)

3. Navigate to the **Provider Apps** section.

4. Select **GoToWebinar** from the Provider list.

5. Enter the previously obtained **API Key** in the **API Key** field.

   ![Ampersand Project Selection](/images/provider-guides/goto-amps.gif)

6. Click **Save Changes**.
